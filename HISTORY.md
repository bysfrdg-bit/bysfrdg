# 🏎️ KartPy 2.5D 개발 기록 (History)

## 1. 프로젝트 개요
- **목표:** 슈퍼 패미컴(SFC)의 '마리오 카트' 스타일인 Mode 7(Pseudo-3D) 방식을 현대적인 Python 기술로 재현.
- **개발 환경:** Windows, Python 3.14
- **주요 라이브러리:** `pygame-ce` (렌더링), `numpy` (고속 픽셀 연산), `math` (삼각함수 물리)

## 2. 주요 구현 기능

### 🏁 Pseudo-3D (Mode 7) 렌더링 엔진
- **알고리즘:** 화면 하단의 바닥 영역을 수평선 방향으로 한 줄씩 스캔하며 거리에 따른 원근감을 계산.
- **최적화:** Python의 느린 루프 대신 `NumPy`의 벡터화 연산을 사용하여 실시간 60FPS 렌더링 달성.

### 🏎️ 드라이빙 및 드리프트 시스템
- **가속/제동:** 관성과 마찰력을 고려한 자연스러운 물리 엔진.
- **Shift 드리프트:** 회전 중 `Shift` 키를 눌러 미끄러지며 드리프트 게이지를 충전.
- **부스트:** 드리프트 종료 시 충전된 게이지에 따라 순간적인 속도 증폭(Boost) 발동.

### 🌲 빌보드 오브젝트 (Billboard System)
- **오브젝트:** 나무(장애물), 아이템 박스, AI 경쟁자.
- **렌더링:** 2D 스프라이트를 항상 플레이어를 향하게 하고, 거리에 따라 크기를 조절하여 3D 깊이감 표현.
- **정렬:** 'Painter's Algorithm'을 적용하여 먼 오브젝트부터 그려 겹침 현상을 해결.

### 🤖 AI 경쟁자 (NPC)
- **주행 로직:** 트랙 중심을 축으로 원형 경로를 따라 자동 주행.
- **개성:** 각 AI마다 고유한 색상, 속도, 주행 라인을 부여하여 다채로운 레이싱 경험 제공.

## 3. 조작 방법 (Controls)
| 키 | 기능 |
| :--- | :--- |
| **방향키 (상/하)** | 가속 및 후진 |
| **방향키 (좌/우)** | 핸들 조작 (조향) |
| **L/R Shift** | 드리프트 (회전 중 꾹 누르기) |
| **Shift 해제** | 부스트 발동 (게이지 충전 시) |

## 4. 소스 코드 요약
- **`main.py`**: 전체 게임 로직, 렌더링 엔진, 물리 계산이 포함된 핵심 파일.
